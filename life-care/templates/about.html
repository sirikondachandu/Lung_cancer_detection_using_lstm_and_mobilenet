Lung Cancer Detection

A brief about the Project

The MobileNet-LSTM hybrid model for lung cancer detection combines the strengths of MobileNet, a lightweight convolutional neural network (CNN) designed for efficient image classification, with Long Short-Term Memory (LSTM) layers, which are well-suited for sequential data processing. In this approach, MobileNet is used to extract features from lung images, capturing the spatial characteristics of the data. These features are then passed to the LSTM layer, which helps in modeling complex patterns or temporal dependencies in the data. The model classifies the lung images into three classes: Benign cases, Malignant cases, and Normal cases, aiding in the early detection of lung cancer. This hybrid approach provides an efficient and accurate solution for classifying medical images with limited computational resources.
